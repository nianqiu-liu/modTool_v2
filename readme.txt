一、软件名称：[game]mod管理工具,其中[game]部分为config文件中的game字段

二、文件说明
2.1：配置文件
    文件名：config.txt
    内容共三行：
        第一行为[game]字段，表示游戏的名称
        第二行为[mod_dir]字段，表示可用mod的源路径
        第三行为[game_dir]，表示mod需要放置的游戏目标路径
2.2：[mod_dir]文件夹，存放所有可用mod
    其结构及说明如下：
        [mod_dir]，mod的根目录
            |-[mod文件夹]，具体mod的文件夹，有多个，不可重名
                |-[预览图]，所在子文件夹对应的mod的预览图，可以有多张
                |-[mod文件]，所在子文件夹对应的mod的具体mod文件，可以有多个
